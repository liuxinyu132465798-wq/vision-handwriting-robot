# 基于视觉的自主临摹书写机器人设计

## 项目简介
本项目为本科毕业设计上位机软件部分，实现图像输入、图像预处理、骨架提取、路径生成、路径排序优化、不断笔连接以及 G-code 输出等功能。

## 主要功能
- 本地图片导入
- 摄像头采集
- 图像预处理
- 骨架提取
- 路径排序优化
- 不断笔连接
- G-code 输出
- InkSight 扩展调用

## 运行环境
- Python 3.x
- OpenCV
- NumPy
- Pillow
- Tkinter

## 使用说明
1. 启动主程序
2. 导入图片或采集图像
3. 执行图像处理与路径生成
4. 导出 G-code 文件
5. 在控制软件中执行输出文件
